# 建房奴隶

最后修改于 2022.5.8



我在黎明镇没待多久， **sadiao** 就要求我建一个黎明镇的集体宿舍（城镇不给大家领地皮，可笑），但我还是答应了——不然我肯定会被排挤到天涯！ **sadiao** 给了我10000K币，叫我开始建。

我断断续续建了12个小时，这个宿舍被艰难的建好了。我的肝非常疼，因为我从来没有熬过夜，但却没有一个人说过一句（哪怕一句）辛苦了。在我建好宿舍的时候，是黎明镇最受欢迎的时候。包括现在PaFoo的一些人都入住了自己的房间。这是后话。

其间我看到了一个人（这就是命运吧），他的皮肤是一个潜水员，衣服是黄色的潜水衣，看着就像一个萌新。我专门带领他到自己的房间里，还教他用箱子锁锁门。他看着很好相处，一路跟着我走。哦对了，他的名字叫 **yh_mc** ，好像还有个朋友叫做 **innovationn** 。我有意去记住这两个名字。

我入服时大概是1月15日，马上就要到春节了。在春节，我是不能玩电脑的。于是，我离开了2服几天。

我在2服待的这15天时间里，也有了许多自己的矿粉，各种东西即使再破烂，毕竟都是我自己的心血。 **yh_mc** 来黎明镇两三天就走了，在黎明镇对面建了一个血色镇。这个镇子是我眼看着创立的。黎明镇也拥有了自己的反应堆。但是，我没有想到的是，在春节的这几天，一场厄运正在我身上发生……
